## Workflow Rules

### 1. Nueva tarea identificada
```bash
# Mover a Backlog (siempre entra por abajo)
curl -s -X PUT "https://api.trello.com/1/cards/{CARD_ID}?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN&idList=69c6e325bf17677b163fa4ea&pos=bottom"
```

### 2. Comenzar tarea (Backlog → To Do → Doing)
```bash
# Mover a To Do (prioridad)
curl -s -X PUT "https://api.trello.com/1/cards/{CARD_ID}?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN&idList=69cd13bd5e2c80944f6edffe&pos=top"

# Cuando arrancas a trabajar
curl -s -X PUT "https://api.trello.com/1/cards/{CARD_ID}?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN&idList=69cd13c1fa7c18235af9de54&pos=top"
```

### 3. Crear branch para tarea
```bash
git checkout -b feat/{TICKET_ID}-{short-description}
```

### 4. Subir PR (Doing → Review)
```bash
git push -u origin feat/{TICKET_ID}-{short-description}
# Luego mover la tarjeta a Review y linkear el PR
```

### 5. Merge/Completar (Review → Done)
```bash
# Mover a Done y agregar comment con PR link
curl -s -X PUT "https://api.trello.com/1/cards/{CARD_ID}?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN&idList=69c6e2e429a87a78c3c9b415"

# Agregar comentario con detalles
curl -s -X POST "https://api.trello.com/1/cards/{CARD_ID}/actions/comments?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN" \
  -d "text=✅ Completado! PR: {PR_URL}"
```

## Common Operations

### Ver todas las tarjetas
```bash
curl -s "https://api.trello.com/1/boards/Ubc9FwJL/cards?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN" | jq '.[] | {id, name, idList}'
```

### Ver tarjetas de una lista
```bash
curl -s "https://api.trello.com/1/boards/Ubc9FwJL/cards?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN" | \
  jq '[.[] | select(.idList == "LIST_ID")]'
```

### Mover tarjeta
```bash
curl -s -X PUT "https://api.trello.com/1/cards/{CARD_ID}?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN&idList={DEST_LIST_ID}"
```

### Agregar comentario
```bash
curl -s -X POST "https://api.trello.com/1/cards/{CARD_ID}/actions/comments?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN" \
  -d "text={TU_COMENTARIO}"
```

### Crear nueva tarjeta
```bash
curl -s -X POST "https://api.trello.com/1/cards?key=$TRELLO_API_KEY&token=$TRELLO_TOKEN" \
  -d "name={TAREA}&idList={LIST_ID}&pos={top|bottom}"
```
